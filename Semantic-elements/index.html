Semantic-Elements

Semantic elements are a type of HTML language for HTML5 to outline different parts of a web page. This newer type of HTML language is helpful in many ways. The list of elements clearly describes each part of a web page. This makes it easier for browsers to read the HTML pages, and so most web browsers support HTML5 semantic elements. These websites are easier to search and program. It also helps organize your document so that you can clearly define different parts of the outline of your page. Semantic elements can do anything from change the color of your web page to add pictures, and articles. Also, because it is more inclusive and universal there are ways to make websites adapt to different countries. It can help change things such as the language, time zones, and the direction of the wording. Semantic elements is a great way to build a web page.
